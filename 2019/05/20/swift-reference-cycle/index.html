为什么会变成这样呢？

Swift 中的引用方式有三种：强引用、弱引用（weak）和无主引用（unowned）。具体的可以看《The Swift Programming Language》里关于 Automatic Reference Counting（自动引用计数）的介绍，不过简单总结一下的话就是

  • 强引用：默认，抓着其他对象不放，直到自己的生命结束
  • 弱引用：佛系引用，对象在就有值，对象没了就是 nil，所以必须是可选类型
  • 无主引用：强制解包的弱引用，对象在一切安好，对象没了直接崩溃
